## Projets phares

 **Projet de modélisation prédictive - Détection de Fraude Carte de Crédit


Dans ce projet, j’ai développé un modèle de machine learning pour analyser des données structurées et produire des prédictions.

Étapes principales :
- Nettoyage et préparation des données
- Analyse exploratoire approfondie
- Gestion du déséquilibre des classes
- Comparaison de plusieurs modèles
- Évaluation avec ROC-AUC et PR-AUC
- Sélection du meilleur modèle

Résultat :
Le modèle retenu améliore significativement la capacité de détection tout en maintenant un bon équilibre précision/rappel.

Impact :
Ce type de modèle peut aider une organisation à mieux cibler ses décisions stratégiques et réduire les risques liés à de mauvaises prédictions.

👉 Voir le projet complet dans mes repositories : https://github.com/Ahcene17/detection-fraude-carte-credit
